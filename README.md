# Voice Dictation - Code

Streaming speech-to-text (ASR) voice dictation for Visual Studio Code.

Vendor-neutral by design. Tested with Doubao streaming ASR (`volc.seedasr.sauc.duration`).

## Get Started

1. Configure credentials in settings (`Cmd+,` → search `voice-dictation-code`).
2. Click the **Dictate** icon in the status bar to start (recommended: use the keyboard shortcut, Default Mode: `Ctrl+Cmd+V` on Mac / `Ctrl+Alt+V` on Win/Linux; Stream Mode: `Ctrl+Cmd+Shift+V` / `Ctrl+Alt+Shift+V`).

## Development

```bash
pnpm install
```

F5 to launch.
